AXL OTel demo

Peer-to-peer distributed tracing for an AXL agent mesh. Every span emitted by every peer flows back to the originator's Jaeger via a tiny OTel sidecar — no central collector, no preconfigured peer list.

What you're looking at

This page is the Editor — the originator. When you submit a prompt it asks Claude Sonnet 4.6 which mesh tools to call, then dispatches MCP requests to other peers across the AXL P2P network. Each downstream peer can call further peers; every span lands here in this Editor's local Jaeger thanks to the originator_peer_id baggage entry the OTel sidecar routes on.
